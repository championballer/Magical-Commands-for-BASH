# spellcmds
Linux terminal commands replaced with harry potter spells

1. To xkill process:
```
avadakedavra
```
2. To clear screen:
```
obliviate
```
3. To change directories:
```
apparate $1
```
4. To exit terminal:
```
knox
```
5. To copy files:
```
horcrux $1 $2
```
6. To list files:
```
lumos
```
7. To view a single file:
```
revelio $1
```
8. To execute sudo commands
```
alohomora $@
```
